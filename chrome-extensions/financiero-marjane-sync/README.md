# Financiero Marjane Sync

Extension Chrome locale pour recuperer une liste Marjane depuis une session navigateur deja connectee.

## Installation

1. Ouvrir `chrome://extensions`.
2. Activer `Mode developpeur`.
3. Cliquer `Charger l'extension non empaquetee`.
4. Choisir le dossier `chrome-extensions/financiero-marjane-sync`.
5. Ouvrir un onglet `https://www.marjane.ma` et se connecter normalement.
6. Ouvrir Financiero sur `/integrations/marjane`.
7. Cliquer `Recuperer liste Marjane`.

L'extension ne demande pas et ne stocke pas d'identifiant, mot de passe, cookie, session ou token Marjane.
